# 圖書租借系統

## 系統功能
- 新增書籍
- 註冊會員
- 借書
- 還書
- 查看可借閱書籍
- 查看會員已借書籍

## 使用說明
1. 確保已安裝Python 3.x
2. 運行 `python library_cli.py` 啟動系統
3. 根據菜單提示進行操作

## 主要模組
- `library_management.py`: 核心邏輯實現
- `library_cli.py`: 命令行界面

## 範例操作
1. 選擇 1 新增書籍
2. 選擇 2 註冊會員
3. 選擇 3 借書
4. 選擇 4 還書
5. 選擇 5 查看可借閱書籍
6. 選擇 6 查看會員已借書籍
7. 選擇 0 退出系統
